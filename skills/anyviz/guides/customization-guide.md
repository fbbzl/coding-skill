# 自然语言美学定制指南

调用者可以通过自然语言描述来覆盖默认美学规范。本指南定义了自然语言到美学参数的映射规则。

---

## 整体风格定制

### 预设主题快速切换

| 自然语言                             | 实际效果                                    |
|-------------------------------------|-------------------------------------------|
| "用深色主题" / "暗色模式"             | 背景 #1A1A1A，文字 #E8E8E8，色板亮度 +15%   |
| "学术风格" / "论文风格"               | 字体 → serif，网格线去除，边框保留            |
| "极简风格" / "简洁一些"               | 去除网格线，去除边框，仅保留轴线和数据          |
| "商务风格" / "企业报告"               | 默认主题（已经偏向商务），增加 logo 空间       |
| "彩色/活泼"                         | 使用 multi_hue sequential，分类色板饱和度 +20% |
| "黑白/灰度"                         | 色板替换为灰度梯度，线条使用不同 dash 模式区分  |
| "现代/科技感"                       | 圆角柱状图（radius: 4px），渐变色，细字体       |
| "手绘风格"                          | 使用 xkcd 或 rough.js 风格的渲染              |

### 颜色定制

| 自然语言                             | 映射操作                                    |
|-------------------------------------|-------------------------------------------|
| "主色用蓝色"                         | categorical[0] = 指定蓝色                    |
| "用暖色调"                           | 色相限制 0-60°（红橙黄）                      |
| "用冷色调"                           | 色相限制 180-270°（青蓝紫）                   |
| "颜色对比强一些"                      | 相邻色相间距 ≥ 30°                           |
| "颜色柔和"                           | 饱和度 -20%                                 |
| "用渐变蓝"                           | sequential 使用 Blues 单色相色板              |
| "用品牌色 #FF6600"                   | categorical[0] = #FF6600，其余用互补色生成     |
| "百分比用绿色，亏损用红色"              | 正值 → #3ca951，负值 → #ff725c               |
| "色盲友好"                           | 确保当前色板通过色盲模拟测试（默认已满足）       |

### 排版定制

| 自然语言                             | 映射操作                                    |
|-------------------------------------|-------------------------------------------|
| "字体大一些"                         | 所有字号层级 ×1.15                           |
| "标题更突出"                         | h1 字号 +2px, 字重 → 700                     |
| "用衬线字体"                         | 字体族 → serif                              |
| "用等宽字体"                         | 字体族 → monospace                          |
| "中文字体优先"                        | font-family 开头加入 PingFang SC / Microsoft YaHei |
| "紧凑排版"                           | 行高 ×0.85, 间距 ×0.8                       |
| "宽松排版"                           | 行高 ×1.15, 间距 ×1.2                       |

### 布局定制

| 自然语言                             | 映射操作                                    |
|-------------------------------------|-------------------------------------------|
| "适合手机看"                         | 应用 mobile 断点参数，宽高比 → 1:1           |
| "适合大屏"                           | 宽高比 → 2.0, 字号 +10%                     |
| "正方形图表"                         | 宽高比 → 1:1                                |
| "宽屏展示"                           | 宽高比 → 2.0                                |
| "紧凑一些/空间有限"                   | 边距 ×0.75                                  |
| "留白多一些"                         | 边距 ×1.3                                   |

### 图表元素定制

| 自然语言                             | 映射操作                                    |
|-------------------------------------|-------------------------------------------|
| "显示数据标签"                       | 开启 data labels（字号 9px, 颜色 #888888）    |
| "隐藏图例"                           | legend.show = false                        |
| "去掉网格线"                         | grid.show = false                          |
| "加粗线条"                           | line.width ×1.5                            |
| "圆角柱状图"                         | bar.corner_radius = 4                      |
| "不要动画"                           | animation = false                          |
| "显示数值" / "标注数据点"              | 开启 point markers + data labels            |

## 复合定制（多层修改）

当一句自然语言涉及多个美学属性时，同时应用所有映射。

```
"深色商务主题，主色用橙色"

→ 背景 #1A1A1A，文字 #E8E8E8
→ categorical[0] = #f5a623（或指定橙色）
→ 其余保持商务风格的排版和间距
```

```
"极简风格，适合手机，图表用正方形"

→ 去除网格线和多余边框
→ 应用 mobile 断点参数
→ 宽高比 → 1:1
```

## 定制优先级

1. 调用者明确指定的属性 → 最高优先级
2. 预设主题指定的属性（如"深色主题"影响的背景色、文字色）
3. 默认美学规范（default.json）→ 最低优先级，被上述任何层级覆盖

## 定制的一致性传递

当调用者定制了某个属性后，该定制应传递到同一输出的所有图表：

```
如果调用了"主色用蓝色"，那么：
- 图表 A 的柱状图 → 蓝色
- 图表 B 的折线图 → 蓝色
- 图表 C 的散点图 → 蓝色
所有图表的图例、标题线条等主色元素也跟随变化。
```
