# 智慧能源物联监控中心

## 场景

为新能源企业打造专业级、实时互动的能源监控大屏，支持 IoT 传感器网络、多能源混合系统、设备拓扑管理和告警联动。适用于园区级微网、分布式光伏、风电场、储能中心等多源协同场景。

## 数据与设计意图

| 区块 | 数据类型 | 分析意图 | 图表类型 | 维度 |
|------|--------|--------|--------|------|
| **KPI 卡片行** | 标量 + sparkline | 一览关键指标 | 5 卡 + 迷你折线图 | 发电量、负荷、在线率、碳排、告警 |
| **功率曲线** | 时间序列 | 趋势监控、供需平衡 | 多折线 + 分层面积图 | 光伏/风电/负荷/储能（24h） |
| **能源结构** | 类别占比 | 资源配置、来源透明 | 环形图（donut） | 光伏 42% / 风电 28% / 储能 18% / 市电 12% |
| **系统关键指标** | 标量（0-100）| 运行状态实时反馈 | 3 仪表盘（gauge） | 负载率、电压、温度 |
| **区域能耗对比** | 类别 × 数值 | 区域对标、负荷均衡 | 横向条形（gradient）| 6 区域 + 消耗量 |
| **24h 热力图** | 日历 grid（日 × 小时） | 峰谷识别、用能规律 | 热力图（heatmap） | 72 点阵（7 天 × 24 小时） |
| **告警滚动列表** | 事件流（优先级 + 时间）| 故障响应、运维协同 | 列表（level 着色）| 5 告警样本 |
| **设备拓扑图** | 网络图（节点 + 边）| 系统架构、故障定位 | 力导向图（force layout） | 12 节点 + 13 连接 |
| **告警分布** | 类别计数 | 系统健康度评估 | 环形图 | 正常/告警/严重 |
| **能效排名榜** | 排序数值 | 对标激励、运维优化 | 横向条形（排名色码）| 5 设施 + 能效指标 |

## 美学决策

### 主题：深色科技大屏

- **页面背景**：#03050C（深海蓝）— 仿 SCADA 黑屏，降低眼疲劳
- **卡片背景**：#0b0f1a（略浅深蓝） + rgba(255,255,255,0.06) 1px 描边 — 悬浮感
- **卡片圆角**：12px — 现代感，同时不失严肃
- **文字主色**：#F8FAFC（极亮白） — 高对比度，适合长时间监看
- **文字辅色**：#9696a0（中灰） — 次级信息，不抢眼

### 品牌色板严格遵循

所有图表色彩取自 **anyviz 标准分类色盘**，按类别和含义一致性编排：

| 色值 | 含义 | 用途 |
|-----|------|------|
| #4269d0 蓝 | 主、信任、能量来源 | 光伏发电、负载率、主系列 |
| #3ca951 绿 | 正常、在线、生长 | 在线率、正常告警、效率优秀 |
| #ff725c 珊瑚红 | 严重、故障、损失 | 严重告警、故障设备 |
| #a463f2 紫 | 辅助、变压器、特殊 | 变压器、设备分类 |
| #efb118 琥珀 | 告警、注意、低效 | 告警告警级、告警事件 |
| #6cc5b0 青 | 储能、辅助平衡 | 储能系统、补充能源 |
| #9696a0 灰 | 中立、市电、参考 | 市电、灰度参考线 |

### 状态语义色

- **正常**：#3ca951（绿） — KPI 上升、设备在线、告警已清
- **告警**：#efb118（琥珀） — 阈值越线、需要关注
- **严重**：#ff725c（珊瑚红） — 故障、系统异常、需立即处理

### 视觉效果

1. **辉光渐变**：
   - 功率曲线面积图：从色值 30% 透明度渐降到 0%
   - 仪表盘背景线：从绿过渡到琥珀、珊瑚红
   
2. **阴影与描边**：
   - 卡片描边：1px rgba(255,255,255,0.06)
   - 无卡片阴影（贴紧深色背景，清爽）
   
3. **动画**：
   - ECharts 初始动画 1000ms cubicOut
   - 实时数据更新：每 3s 闪烁一次 KPI 值（±随机方差）
   - LIVE 指示灯：持续脉冲（1.5s 周期）
   - 顶部时钟：实时秒数更新

### 布局与响应式

**大屏设计尺寸**：1920×1080（16:9 宽屏）

**1920px 标准布局**：
- 顶部固定标题栏：高 70px
- KPI 行：5 列均匀分布（grid 5fr）
- 图表网格：5 列基础，支持跨列
  - 功率曲线：跨全 5 列（full-width）
  - 环形图 + 仪表板：各占 2.5 列（half）
  - 区域对比：跨全 5 列
  - 热力图 + 告警列表：各占 2.5 列
  - 拓扑图：跨全 5 列
  - 告警分布 + 效率排名：各占 2.5 列

**响应断点**：
- **1440px 及以下**：grid 4 列，卡片高度略小
- **1024px 及以下**：grid 2 列，mobile 适配
- **640px 及以下**：grid 1 列，font scale 85%

## 技术栈

- **库**：ECharts 5.5.1（企业级交互式图表库）
- **环境**：Web 前端（HTML5 + JavaScript）
- **输出**：完全自包含单文件 HTML（无构建步骤，无网络依赖除 CDN）
- **性能**：8 个独立 ECharts 实例 + DOM list（总计 <2MB，<100ms 首屏）

### ECharts 配置原则

1. **基础配置**：
   ```javascript
   {
     backgroundColor: 'transparent',
     textStyle: { fontFamily: FONT, color: '#F8FAFC' },
     animation: true,
     animationDuration: 1000,
     animationEasing: 'cubicOut'
   }
   ```

2. **所有颜色参数写死**：不依赖 ECharts 默认色盘，所有 series/itemStyle 明确指定

3. **所有尺寸参数写死**：fontSize、lineWidth、padding、margin 全部是确定的像素值或比例

4. **Tooltip 格式一致**：所有 tooltip 用相同背景、边框、字样

## 运行方式

1. **直接打开**：在现代浏览器中打开 `index.html` 文件（推荐 Chrome 100+、Safari 15+、Firefox 95+）
2. **无需构建**：所有资源都是内联的，无需网络（除非 CDN 无法访问，则失败）
3. **全屏展示**：按 F11 全屏，大屏显示效果最佳
4. **响应式缩放**：窗口缩放时自动调整，所有图表响应 window.resize 事件

### 注意事项

- 大屏最佳显示尺寸：**1920×1080** 或更高分辨率
- 小屏兼容：支持向下缩放至 1024px，但信息密度会降低
- 性能：实时更新 KPI 频率 3s 一次，不会造成卡顿
- 数据：所有数据是模拟的，适合演示和截图；如需真实数据，修改 JavaScript 中的数据源即可

## anyviz 设计决策说明

### 为什么选择这些图表类型？

| 图表 | 理由 |
|------|------|
| **多折线 + 面积** | 时间序列 + 多变量同时比较，能量守恒（供应 vs 消费）直观可见 |
| **环形图** | 占比 + 视觉层级，对称美观，适合 3-4 分类 |
| **仪表盘** | 实时状态一眼看清，工业 SCADA 标准控件 |
| **横向条形** | 类别对比且避免 X 轴拥挤，6 个区域不堆压 |
| **热力图** | 峰谷规律一目了然，日历形式符合运维认知 |
| **力导向图** | 网络拓扑自然布局，避免手工排列线路交叉 |

### 为什么采用深色主题？

1. **工业规范**：SCADA 系统传统黑屏，降低长时间盯看的眼疲劳
2. **大屏环境**：深色背景在投影仪、LED 屏上亮度均衡
3. **对比度**：极亮白字在暗背景上对比度 21:1，易阅读
4. **品牌一致**：与 anyviz "AI-native 现代数据设计" 理念吻合

### 为什么限制在 7 色？

1. **感知容量**：人眼最多区分 7±2 个饱和色，超过则混淆
2. **一致性**：anyviz 色盘本身是经过 ColorBrewer 优化的
3. **颜色盲友好**：7 色在绿色盲、红色盲者眼中仍可区分
4. **专业感**：克制用色显示高端定制

## 扩展建议

如需集成真实数据，修改以下部分：

1. **数据源替换**：将 hardcoded 的 `const data = [...]` 替换为 `fetch('/api/energy')` 获取动态数据
2. **实时推送**：使用 WebSocket 或 Server-Sent Events 推送实时更新
3. **时间轴滚动**：修改功率曲线 X 轴，支持任意 24h 窗口滑动
4. **告警联动**：点击告警列表项，高亮对应设备在拓扑图上
5. **导出功能**：ECharts 内置 saveAsImage()，支持截图/PDF 导出
