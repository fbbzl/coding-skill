# 城市大数据可视化中心

## 场景描述

本示例是一个典型的**智慧城市指挥中心大屏**，面向城市运营者、数据分析师和管理部门，展示跨地域的人口、流量、经济指标等多维度数据。主要应用场景包括：

- **城市规划部门**：监控常住人口、城市发展趋势
- **交通运输部门**：实时客流监测、流量热力分析
- **商业零售**：城市间销售排名、地区分布对比
- **公共安全**：应急响应、人群密度监控
- **旅游文化**：景区客流、城市间旅游流向

## 技术栈

- **地图库**：ECharts 5.5.1 内置中国地图（含降级方案）
- **图表类型**：
  - Choropleth（省份着色）+ Lines（飞线）+ EffectScatter（涟漪散点）
  - Bar Chart（条形排名）
  - Pie Chart（环形占比）
  - Line Area Chart（趋势面积图）
  - Symmetric Bar Chart（对称条形图）
  - Heatmap（热力图）
- **响应式**：CSS Grid + Window resize 事件
- **无构建**：单文件 HTML，所有 CSS/JS 内联

## 图表清单

| 序号 | 组件 | 图表类型 | 数据维度 | 设计意图 |
|------|------|---------|---------|---------|
| 1 | 顶部 KPI 卡片（5 个） | 指标卡 + 数值 | 人口、GDP、客流、城市数、实时活跃 | 一眼掌握全局关键指标 |
| 2 | 中央大地图 | Choropleth + Lines + EffectScatter | 省份客流、飞线分发、重点城市 | 地理空间视觉化，核心信息中央聚焦 |
| 3 | 城市排名卡 | Bar List（排名纵向列表） | TOP 10 城市客流量 | 快速定位高价值城市 |
| 4 | 区域占比 | Donut / Pie Chart | 华东/华南/华北/西部/东北 | 区域分布一目了然 |
| 5 | 客流趋势 | Line Area Chart | 近 30 天日均客流 | 识别流量增减趋势 |

## 运行方式

1. 在浏览器中直接打开 `index.html`，无需构建或服务器
2. 页面将自动初始化所有图表，实时时钟每秒更新
3. **地图降级方案**：
   - 首先尝试加载中国地图（`echarts.getMap('china')`）
   - 若地图脚本不可用或加载失败，自动降级为**经纬度散点图**
   - 散点大小代表该城市客流量，不会白屏

## anyviz 品牌色与设计决策

### 色板（按优先级使用）

| 色值 | 名称 | 用途 |
|------|------|------|
| `#4269d0` | 蓝 | 主图表线条、地图 choropleth 基调、KPI 排名序号 |
| `#3ca951` | 绿 | 飞线效果、同比增长指标、"LIVE" 指示灯 |
| `#ff725c` | 珊瑚红 | 警告/异常指标 |
| `#a463f2` | 紫 | 区域占比色、补充色 |
| `#efb118` | 琥珀 | 中性高亮、金融数据 |
| `#6cc5b0` | 青 | 飞线梯度终点、人口性别（女性） |
| `#9696a0` | 灰 | 次要文本、border、分隔线 |

### 深色大屏美学

- **背景**：`#03050C`（接近纯黑，显示器发热最低）
- **卡片面板**：`#0b0f1a` 或 `rgba(255,255,255,0.02)`（透明度叠加）
- **边框**：`1px solid rgba(255,255,255,0.06)`（极淡，保持层级感）
- **圆角**：统一 `12px`（现代科技感）
- **文字**：
  - 主文本 `#F8FAFC`（几乎白色，高对比度）
  - 次要文本 `#9696a0`（灰色，降低视觉权重）
- **字体**：`'Helvetica Neue', Helvetica, Arial, 'PingFang SC', 'Microsoft YaHei', sans-serif`（跨平台）
- **数字等宽**：`'Menlo', 'Monaco', monospace`（对齐和精确感）
- **辉光**：克制使用，仅在地图涟漪和飞线动画中应用 `box-shadow` 和 `opacity`

### 地图飞线设计

- **起点**：北京（中心枢纽）`#4269d0` 蓝色
- **流向**：9 条飞线向外发散，`opacity: 0.6`
- **梯度**：飞线效果中蓝 → 青 → 绿，象征信息流向和能量传递
- **动画**：`period: 5s`，符号 `arrow`，流动感强烈但不刺眼

## 地图降级方案说明

### 正常流程

```javascript
// 尝试使用中国地图
if (echarts.getMap('china') !== undefined) {
  // 加载 choropleth + lines + effectScatter
}
```

### 降级逻辑

若地图脚本加载失败或地图不存在，自动切换为**经纬度散点图**：

```javascript
try {
  // 初始化地图
} catch (error) {
  console.warn('地图加载失败，降级为散点图');
  initFallbackScatter(mapChart);
}
```

**降级后的表现**：

- 使用 `xAxis` 和 `yAxis` 定义经纬度范围（中国地理范围）
- 15 个主要城市以散点形式展示，大小代表客流量
- 城市名称标签显示在右侧
- 保持蓝色主题色，无损用户体验

## 数据来源说明

所有数据为**模拟生成**，用于演示目的：

- **人口数据**：基于真实中国城市常住人口量级（千万级）
- **GDP**：基于真实经济总量量级（万亿级）
- **城市坐标**：采用真实的主要城市经纬度（如北京 116.4°E, 39.9°N）
- **趋势数据**：随机生成但保持合理范围
- **热力数据**：24 小时 × 7 天矩阵，随机分布
