# 电商运营数据中心

**场景**：大型电商零售平台的实时运营监控大屏，适用于运营中心、数据分析部门、高管办公室的实时决策场景。

## 快速开始

1. 在浏览器中打开 `index.html`（无需本地构建、无依赖安装）
2. 大屏自动加载，实时时钟与模拟数据动态更新

## 技术栈

| 层级 | 技术 |
|------|------|
| 图表库 | ECharts 5.5.1（CDN 加载） |
| 地图 | China GeoJSON（ECharts 原生地图，含降级方案） |
| 布局 | CSS Grid + Flexbox，支持 1920×1080 / 平板 / 手机自适应 |
| 样式 | 深色主题（#03050C 背景），品牌色板，辉光与渐变效果 |
| 交互 | 原生 JavaScript + ECharts 事件，鼠标悬停高亮 |

## 大屏内容清单

### 1. 顶部标题栏（固定）
- **左侧**：anyviz 品牌标识（渐变文字）+ 标题"电商运营数据中心"
- **右侧**：实时时钟 + LIVE 绿色脉搏指示灯
- **设计**：毛玻璃效果、半透明背景、深色科技感

### 2. KPI 卡片行（第 1 行，5 列）

| 指标 | 当前值 | 同比 | 环比 | 迷你图 |
|------|--------|------|------|--------|
| 当日 GMV | ¥8,240 万 | ↑ 12.5% | - | 上升趋势 |
| 订单数 | 1,264 件 | - | ↑ 8.3% | 上升趋势 |
| 客单价 | ¥652 | - | ↑ 3.8% | 上升趋势 |
| 转化率 | 3.24% | - | ↓ 0.2% | 下降趋势 |
| 退款率 | 1.8% | - | ↓ 0.1% | 下降趋势 |

每卡片配迷你折线图（Sparkline）、同环比箭头、绿色上升/红色下降。

### 3. 销售趋势图（第 2 行左，占 3 列）
- **图表类型**：双轴折线+柱 (mixed chart)
- **数据**：近 7 天数据
- **左轴**：订单数（柱，蓝色，不透明）
- **右轴**：GMV 销售额（线，珊瑚红，带渐变面积填充）
- **意义**：关联订单增长与销售额趋势，发现偏离

### 4. 品类销售分布（第 2 行右，占 2 列）
- **图表类型**：环形图（Donut）
- **数据**：6 大品类（服装箱包、美妆个护、食品酒水、家居日用、电子产品、其他）
- **色板**：品牌色板依次（蓝→绿→珊瑚红→紫→琥珀→青）
- **标签**：环内直接标注百分比，无图例混淆
- **意义**：一眼看出品类策略效果、主力品类依赖度

### 5. 区域销售热力（第 3 行左，占 3 列）
- **图表类型**：中国地图 Choropleth（颜色渐变），或气泡散点（降级方案）
- **数据**：各省销售额 TOP 15
- **色板**：顺序蓝系（#05468a → #c5dff8），符合 anyviz 无障碍规范
- **交互**：鼠标悬停高亮、显示省份名 + 销售额
- **意义**：一目了然发现区域热点、冷区，指导营销投入

#### 地图降级方案
若中国地图加载失败（网络/权限），自动切换为**气泡散点地图**：
- 纬度/经度基于各省政治中心近似坐标
- 气泡大小与销售额成正比
- 蓝色气泡、黑色边框、半透明，防止重叠遮挡

### 6. 转化漏斗（第 3 行右，占 2 列）
- **图表类型**：Funnel
- **路径**：曝光 → 点击 → 加购 → 下单 → 支付
- **每级转化率**：曝光 100% → 点击 72% → 加购 48% → 下单 32.4% → 支付 31.48%
- **色板**：品牌色从上到下渐进（蓝→绿→珊瑚→紫→琥珀）
- **意义**：快速定位转化瓶颈（如"点击到加购跳失高"），驱动优化聚焦

### 7. 渠道来源分布（第 4 行左，占 2 列）
- **图表类型**：100% 堆叠横条
- **渠道**：App (38%) / 小程序 (28%) / Web (22%) / H5 (12%)
- **色块内标签**：直接显示百分比，清晰无歧义
- **意义**：渠道优先级排序，决定投入侧重

### 8. 热销 TOP 商品（第 4 行右，占 3 列）
- **图表类型**：横向条形图（left-aligned bar chart）
- **TOP 6 商品**：
  1. 夏季新款T恤 - ¥3,420
  2. 防晒霜 SPF50 - ¥2,860
  3. 运动休闲鞋 - ¥2,540
  4. 无线蓝牙耳机 - ¥2,180
  5. 牛仔裤男款 - ¥1,950
  6. 护肤面膜套装 - ¥1,680
- **渐变色**：蓝→紫梯度填充，右侧标注金额
- **意义**：爆款对标、库存预警、营销推广聚焦

## anyviz 品牌设计决策

### 色彩系统

从 `aesthetics/default.json` 品牌色板严格取色（无例外），确保 UI 一致性：

| 用途 | 颜色 | Hex | 设计意图 |
|------|------|-----|----------|
| 主数据（订单、用户） | 蓝 | #4269d0 | 信任、科技感 |
| 正向指标（增长、同比向好） | 绿 | #3ca951 | 积极、乐观 |
| 告警/负向指标 | 珊瑚红 | #ff725c | 吸引注意、危险 |
| 辅助数据 | 紫 | #a463f2 | 区分、品质 |
| 高亮数字 | 琥珀 | #efb118 | 温暖、强调 |
| 细节 | 青 / 灰 | #6cc5b0 / #9696a0 | 补充、淡化 |

### 深色主题参数

- **页面底色**：#03050C（极深蓝黑，减少眼睛疲劳）
- **卡片背景**：#0b0f1a 或 rgba(255,255,255,0.02)（极低白度，提升对比）
- **边框**：1px rgba(255,255,255,0.06)（精细网格感，不刺眼）
- **圆角**：12px（现代友好）
- **毛玻璃**：backdrop-filter: blur(8px)（高端感）

### 文本与排版

- **主文本**：#F8FAFC（94.6% 灰度，WCAG AAA 对黑背景对比度 > 7:1）
- **次文本**：#9696a0（对黑背景对比度 3.8:1，用于标签/说明）
- **字体**：'Helvetica Neue', Helvetica, Arial, 'PingFang SC', 'Microsoft YaHei', sans-serif
- **标题**：16px / 600 weight / 行高 1.3
- **KPI 大数字**：32px / 700 weight / 等宽字体
- **图表标签**：10-11px / 400 weight / 色彩对比度达标

### 视觉层级

1. **顶层**：标题栏 + KPI 卡片（大字、品牌色）
2. **中层**：主图表（占据大面积、交互）
3. **底层**：图例、数据标签、时间戳（小字、灰色）

### 适度辉光与渐变

- **KPI 卡悬停**：`box-shadow: 0 6px 16px rgba(66, 105, 208, 0.15)`（蓝色微辉）
- **面积图**：线性渐变填充（顶部 30% 不透明度 → 底部 0%，丝滑过渡）
- **TOP 商品条**：蓝→紫梯度（视觉流畅）
- **LIVE 指示灯**：脉搏动画（1.5s 周期，吸引眼球）

克制原则：不超过 2 层渐变、最多 1 个辉光效果/卡片，避免炫彩感。

## 数据说明

所有数据为模拟数据，具有真实电商场景特征：
- **GMV**：千万级（¥7,200 ~ ¥8,350 万/日）
- **订单数**：千级（1,120 ~ 1,310 件/日）
- **客单价**：百元级（¥652 平均）
- **转化率**：3% 量级（符合行业水位）
- **退款率**：< 2%（健康指标）
- **品类分布**：服装、美妆主导（垂直电商特征）
- **区域热力**：一二线城市优先（现实写照）

## 扩展建议

1. **数据接驳**：替换模拟数据为 API 接口（如 `/api/kpi`、`/api/sales-trend`）
2. **实时更新**：WebSocket 推送、或轮询接口（`setInterval(fetchData, 5000)`）
3. **多时间维度**：添加"天/周/月"选项卡，切换数据粒度
4. **导出能力**：截图/PDF 下载、数据 CSV 导出
5. **权限控制**：用户登录、角色-字段可见性
6. **国际化**：i18n 多语言支持（中文 / 英文 / 日文）
7. **主题切换**：浅色主题选项、自定义品牌色
8. **告警规则**：KPI 异常闪烁、推送通知

## 无障碍合规

| 需求 | 实现方式 |
|------|----------|
| 色盲友好 | 品牌色板经 ColorBrewer 验证；同/环比用 ↑↓ 符号冗余编码 |
| 对比度 | 文本 > 4.5:1；数据元素 > 3:1（WCAG AA 达标） |
| 字号可读 | 最小 9px（脚注）→ 32px（大数字）梯度 |
| 动画无危害 | Sparkline 与 LIVE 脉搏无闪烁（≤ 3 Hz） |
| 触摸目标 | KPI 卡片 ≥ 120×120px；按钮 ≥ 44×44px |
| Alt Text | 每图表配对应说明（场景描述中已含） |
