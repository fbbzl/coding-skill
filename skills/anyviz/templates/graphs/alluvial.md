# 冲积图（Alluvial Diagram）

## 适用场景
- 展示多阶段分类数据的流动与变化
- 追踪样本在不同维度间的分类转移
- 对比多个时间点或多维分类的成员变化
- 类别带状流表示：颜色按起始类别，宽度按数量
- 最佳阶段数：2-6 个；最佳分类数：3-12 个

## 数据格式
```json
{
  "dimensions": ["2022年部门", "2023年部门", "2024年部门"],
  "records": [
    { "2022年部门": "销售", "2023年部门": "销售", "2024年部门": "市场", "count": 15 },
    { "2022年部门": "销售", "2023年部门": "技术", "2024年部门": "技术", "count": 25 },
    { "2022年部门": "技术", "2023年部门": "技术", "2024年部门": "管理", "count": 8 },
    { "2022年部门": "技术", "2023年部门": "市场", "2024年部门": "市场", "count": 12 }
  ]
}
```

## 美学参数

```json
{
  "alluvial": {
    "axis_width": 20,
    "axis_color_mode": "by_source",
    "axis_opacity": 0.8,
    "flow_opacity": 0.5,
    "flow_opacity_highlight": 0.8,
    "flow_opacity_deemphasis": 0.15,
    "flow_interpolation": "monotone",
    "category_label_font_size": 10,
    "category_value_font_size": 9,
    "dimension_label_font_size": 11,
    "dimension_label_font_weight": 600,
    "horizontal_spacing": 100,
    "vertical_spacing": 10,
    "color_by_source": true
  }
}
```

## 设计要点

1. **轴与类别编码**：每个维度垂直显示为轴，轴宽 20px；轴上各类别按比例分配高度，对应流量总数；类别标签字号 10px，数值标签 9px，颜色 #555555；维度标题字号 11px，字重 600，位置在轴上方
2. **流带与颜色映射**：流带（带状流）宽度与数量成正比，通过贝塞尔曲线连接相邻轴的对应类别；颜色按起始类别使用 categorical 色板（首色 #4269d0、次色 #3ca951 等），从第一维延续到最后，保持视觉连贯性；基础 opacity 0.5 避免过度绘制
3. **维度间距与流向**：轴间水平距离 100px，足以展示流带曲线；同一轴内类别间垂直距离 10px；流带采用平滑插值（monotone），避免尖锐转折，强调流动感
4. **交互与高亮**：悬停特定类别或流带时，高亮该类别及其所有相关流（opacity 0.8），其他流降至 0.15；支持点击筛选特定源类别，只展示其流向目标；提供流量统计显示（悬停时显示数值和百分比）
5. **无障碍支持**：色盲用户需要形状或图案区分不同起始类别，可用条纹/网格纹理替代纯颜色区分；提供数据表视图，展示每条流的源→目标→目标类别→数量；标签文本完整，不依赖纯颜色识别
6. **异常值与标注**：流量特别大或特别小的流可用注解或箭头标注，帮助用户发现关键变化；支持对数刻度选项以处理数量跨度大的数据

## 变体

### 桑基图（Sankey Diagram）
节点为源和目标，边的宽度表示流量，通常用于展示能量流、资金流等单向流动；与冲积图的主要区别是节点在左右两侧（二部），而非多阶段轴；适合展示起点和终点的流量分配

### 平行集合（Parallel Sets）
类似冲积图，但强调多个维度间的分类关联，用色块和连接线展示类别间的依赖关系；与冲积图的区别是不强调流量大小，而是强调分类结构的完整性
