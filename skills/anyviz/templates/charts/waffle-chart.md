# 华夫图（Waffle Chart）

## 适用场景
- 展示部分与整体的比例关系，每格代表整体的 1%
- 展示多个类别的组成，强调各部分的相对大小
- 相比饼图具有更好的精度对比（人眼更易比较正方形而非扇形）
- 最佳项数：2-5 个类别，总计 100

## 数据格式
```json
{
  "categories": ["产品 A", "产品 B", "产品 C", "产品 D"],
  "values": [35, 25, 20, 20],
  "total": 100
}
```

## 美学参数（从 default.json 继承）

```json
{
  "waffle": {
    "rows": 10,
    "cols": 10,
    "cell_size_px": 16,
    "cell_gap_px": 2,
    "cell_corner_radius_px": 2,
    "cell_stroke_width_px": 1.0,
    "cell_stroke_color": "#FFFFFF"
  }
}
```

## 设计要点

1. **每格单位**：标准 10×10 网格（100 格），每格代表 1%；若数据总和不是 100，需先归一化
2. **排列方向**：默认从左到右、从上到下逐行填充；可根据审美需求改为从下到上
3. **颜色分配**：按 categorical 色板分配各类别色；首类 #4269d0，第二类 #3ca951，第三类 #ff725c，第四类 #a463f2
4. **网格间隔**：单元格间隔 2px，格子尺寸 16px，圆角 2px，白色描边 1.0px 分隔各格
5. **数据标签**：在图例中标注百分比（9px 字体）；或在网格右侧添加类别标签和百分比
6. **无障碍设计**：除颜色外，可添加纹理（hatching 或网格密度变化）区分类别，或为每个类别分块添加序号

## 变体

### 图标华夫图（Pictogram Waffle）
用小图标代替正方形（如人形图标表示人数、房屋图标表示房产比例），直观性更高

### 堆叠华夫图（Stacked Waffle）
多个维度时，网格纵向堆叠，每行代表一个分组，颜色表示子类别

### 非矩形华夫（Hexagon / Circle Waffle）
用正六边形或圆形替代正方形排列，视觉效果更柔和
