# 密度图（Density Plot）

## 适用场景
- 展示单个连续变量的分布形状，特别是非正态分布
- 比较多个分组的分布（分组密度图）
- 识别数据的众数、峰值和离散情况
- 最佳数据点：100-100,000

## 数据格式
```json
{
  "values": [23, 25, 24, 28, 22, 26, 24, 29, 25, 27, ...],
  "groups": [
    {
      "name": "A 类",
      "values": [23, 25, 24, 28, 22, ...]
    },
    {
      "name": "B 类",
      "values": [32, 35, 31, 36, 33, ...]
    }
  ]
}
```

## 美学参数（从 default.json 继承）

```json
{
  "density": {
    "line_width_px": 2.2,
    "line_color": "#4269d0",
    "fill_opacity": 0.12,
    "fill_color": "#4269d0",
    "kernel": "gaussian",
    "bandwidth": "scott"
  },
  "axis": {
    "y_axis": { "label": "密度" }
  }
}
```

## 设计要点

1. **核函数与带宽**：默认采用 Gaussian 核与 Scott 带宽估计；若有领域知识可调整（带宽过小产生噪点，过大过度平滑）
2. **填充与线条**：密度曲线线宽 2.2px，#4269d0；下方填充 opacity 0.12；分组时每组用 categorical 色板对应颜色
3. **Y 轴零点**：密度图的 Y 轴从 0 开始，表示无样本的状态
4. **多分组展示**：分组数 ≤ 3 时可直接叠加，颜色分别取 #4269d0、#3ca951、#ff725c；> 3 时改用分面小倍数
5. **参考线**：可添加中位数/均值参考线（#EE7B6F，虚线 [4,4]，opacity 0.7）
6. **无障碍设计**：多分组时除颜色外加标签或虚实线区分（实线/虚线），确保色盲用户可读

## 变体

### 脊线图（Ridgeline）
多组数据时垂直堆叠，每行一条密度曲线，适合展示 5-20 个分组的分布变化

### 分组密度 + 直方图（Density + Histogram）
在密度曲线下方同时显示直方图（bars, opacity 0.3），结合离散与连续视图

### 二维密度图（2D Density / Contour）
两个连续变量的联合分布，用密度等高线或热力图表示浓度
