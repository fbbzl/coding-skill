# 六边形分箱图（Hexbin Plot）

## 适用场景
- 展示大数据量散点的密度分布（10,000+ 点），避免重叠遮挡
- 用六边形颜色深度表示聚集程度，识别数据热点和冷区
- 替代过度绘制（overplotting）的散点图，保留分布趋势
- 最佳数据点：10,000-1,000,000

## 数据格式
```json
{
  "points": [
    { "x": 3.2, "y": 8.1 },
    { "x": 4.5, "y": 9.3 },
    { "x": 3.1, "y": 8.2 }
  ],
  "bin_size": 0.5
}
```

## 美学参数（从 default.json 继承）

```json
{
  "hexbin": {
    "hexagon_size_px": 8,
    "color_scale": "viridis",
    "color_scale_log": true,
    "fill_opacity": 0.85,
    "stroke_width_px": 0.5,
    "stroke_color": "#E8E8E8",
    "no_data_color": "#F5F5F5",
    "colorbar_label": "点密度"
  }
}
```

## 设计要点

1. **色板与对数映射**：用 Viridis 多色相色板表示密度（低 #440154 深紫 → 高 #FDE725 亮黄）；数据跨度大时用对数色标；无数据区用 #F5F5F5
2. **六边形大小与间隔**：六边形尺寸 8px（可根据分辨率调整），边间隙 0.5px #E8E8E8；密度高的区域六边形颜色深、透明度高（0.85）
3. **分箱策略**：根据数据范围自动选择 bin 尺寸，或用户指定（常见 0.1-0.5 单位）；确保覆盖所有数据点
4. **颜色条**：在图右侧显示 colorbar，标注密度范围和对数刻度，label 字号 10px
5. **参考线**：可添加均值或中位数的参考点（#4269d0，半径 5px）或参考线（#EE7B6F，虚线 [4,4]，opacity 0.7）
6. **无障碍设计**：色盲用户若难以区分，可加透明度梯度（低密度更透明）、或在鼠标悬停时显示具体点数

## 变体

### 方形分箱图（Square Binning / Heatmap）
用方形网格替代六边形，更接近热力图，适合展示规则网格数据

### 密度等高线（Density Contour）
用等高线或等值线圈出密度范围，加 Viridis 色填充，更强调密度峰值

### 2D 直方图（2D Histogram）
用柱体高度表示密度而非颜色，立体感强，但需交互旋转以查看全貌
