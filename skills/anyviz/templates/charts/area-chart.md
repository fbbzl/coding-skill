# 面积图（Area Chart）

## 适用场景
- 展示数据量随时间的变化趋势
- 强调"总量"概念
- 最佳系列数：1-3 个（非堆叠），2-5 个（堆叠）

## 数据格式
同折线图

## 美学参数
```json
{
  "area": {
    "opacity": 0.15,
    "stacked_opacity": 0.7,
    "gradient_top_opacity": 0.3,
    "gradient_bottom_opacity": 0.0
  }
}
```

## 设计要点

1. **单一面积图**：填充 opacity 0.15，叠加折线 2px
2. **堆叠面积图**：不设置透明度，各层使用实色填充以展示结构
3. **重叠面积图**：最多 3 个系列，使用低透明度（0.1-0.15）避免遮挡
4. **Y 轴**：堆叠时从 0 开始；非堆叠时可以不从 0 开始
5. **渐变填充**：优先使用从上到下的透明度渐变（顶部 0.3 → 底部 0.0）
6. **无障碍**：堆叠多层时用白色描边（0.5px）分隔，并直接标注各层名称，避免仅靠相邻填充色区分

## 变体

### 堆叠面积图（Stacked Area）
多系列叠加展示总量及各部分构成，最多 5 个系列，各层实色填充（stacked_opacity 0.65）

### 百分比堆叠面积图（100% Stacked Area）
归一化到 100%，强调各部分相对占比随时间的变化，而非绝对量

### 河流图 / 主题河流（Streamgraph）
堆叠面积的居中变体，基线居中而非底部对齐，适合展示多类别随时间的此消彼长
