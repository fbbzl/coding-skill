# 热力图（Heatmap）

## 适用场景
- 二维矩阵数据的密度/强度展示
- 相关性矩阵
- 时间 × 类别的交叉分析
- 建议行列数：5-30 × 5-30

## 数据格式
```json
{
  "x_labels": ["周一", "周二", "周三", "周四", "周五"],
  "y_labels": ["0-6点", "6-12点", "12-18点", "18-24点"],
  "values": [
    [12, 18, 22, 15, 10],
    [35, 42, 48, 38, 30],
    [55, 68, 72, 60, 48],
    [28, 35, 40, 32, 25]
  ]
}
```

## 美学参数
```json
{
  "color": {
    "sequential": { "name": "Blues" }
  },
  "annotation": {
    "tooltip_font_size": 11
  }
}
```

## 设计要点

1. **色板**：默认使用 Blues 单色相顺序色板；科学数据优先使用 Viridis
2. **网格线**：白色细线（0.5px）分隔单元格
3. **标注**：每个单元格内可标注数值（字号 9px），深色单元格用白色文字
4. **缺失值**：用明显的区分色（如 #EEEEEE）标注
5. **行列排序**：如果没有自然顺序，使用聚类或排序以揭示模式
6. **无障碍**：顺序色板优先 Viridis（明度单调，色盲可读）；单元格内标注数值作为颜色的冗余编码，不让读者仅靠色深判断

## 变体

### 相关系数热力图（Correlation Matrix）
对称矩阵，用 diverging 色板（中心 #F5F5F5 对应 0），展示变量两两相关性，对角线为 1

### 日历热力图（Calendar Heatmap）
按日期网格排布的特例，详见 `calendar-heatmap.md`，适合展示一年内的每日数值

### 聚类热力图（Clustered Heatmap）
行列两侧附树形图（dendrogram），按相似度重排行列，揭示数据块状结构
