# 坡度图（Slope Chart）

## 适用场景
- 展示两个时间点（或两个条件）之间的排名变化或数值变化
- 强调上升趋势（增长）、下降趋势（衰落）及排名交叉
- 对比两个关键时刻的变化（如开始vs结束、方案A vs方案B）
- 最佳项数：5-20 个类别

## 数据格式
```json
{
  "categories": ["产品 A", "产品 B", "产品 C", "产品 D"],
  "start": [120, 85, 95, 110],
  "end": [140, 88, 75, 125],
  "start_label": "2024",
  "end_label": "2025"
}
```

## 美学参数（从 default.json 继承）

```json
{
  "slope": {
    "line_width_px": 2.2,
    "increasing_color": "#3ca951",
    "decreasing_color": "#ff725c",
    "neutral_color": "#9696a0",
    "point_radius_px": 4.5,
    "point_stroke_width_px": 2.0,
    "point_stroke_color": "#FFFFFF",
    "label_offset_px": 8
  }
}
```

## 设计要点

1. **配色与趋势**：上升用 #3ca951，下降用 #ff725c，持平用 #9696a0；确保色盲友好性，配合标签或其他视觉编码
2. **两端标注**：左侧标注开始值和名称（类别 + 数值），右侧标注结束值；字号 10px；标签应避免相互重叠，可根据需要调整垂直位置
3. **连接线与排序**：左端按开始值排序（从上到下），连接线的斜率和交叉表示排名变化；交叉点越多视觉冲击越强
4. **坡度透明度**：若项数多（15+），线条 opacity 可降低到 0.6，高亮项用 opacity 1.0
5. **参考线**：可添加水平参考线表示零增长或平均值（#EE7B6F，虚线 [4,4]，opacity 0.7）
6. **无障碍设计**：用线宽或虚实区分趋势（增长实线粗、下降虚线细），配合颜色和数值标签

## 变体

### 多时间点坡度图（Bump Chart）
展示 3 个或更多时间点的排名演变，用多条连接线显示轨迹，适合长期趋势跟踪

### 宽幅坡度图（Wide Slope）
两端标注的水平间距加大，坡度更平缓，适合展示微小变化

### 分组坡度图（Grouped Slope）
多组数据时，用不同颜色或面板展示各组的坡度变化
