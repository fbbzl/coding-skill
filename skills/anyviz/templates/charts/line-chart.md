# 折线图（Line Chart）

## 适用场景
- 展示数值随时间或其他有序维度的变化趋势
- 对比多个系列的变化模式
- 最佳系列数：1-5 个

## 数据格式
```json
{
  "x": ["2024-01", "2024-02", "2024-03", ...],
  "series": [
    { "name": "产品A", "values": [120, 135, 148, ...] },
    { "name": "产品B", "values": [80, 92, 105, ...] }
  ]
}
```

## 美学参数（从 default.json 继承）

```json
{
  "stroke": {
    "line_width": 2.0,
    "cap": "round",
    "join": "round"
  },
  "point": {
    "line_data_point": { "radius": 3.0, "opacity": 1.0 }
  },
  "axis": {
    "x_axis": { "tick_rotation": 0 },
    "y_axis": { "include_zero": false }
  }
}
```

## 设计要点

1. **Y 轴基线**：如果数据范围远离零（如 120-150），Y 轴不必从零开始，但应标注
2. **数据点标记**：数据点 ≥ 8 时，默认不显示点标记（减少视觉噪音）；数据点较少时显示
3. **颜色分配**：按 categorical 色板顺序分配，第一个系列用 #4269d0
4. **多系列标注**：系列 ≤ 3 时直接标注，> 3 时使用图例
5. **缺失值处理**：如果中间有缺失数据点，用虚线连接断点两侧

## 变体

### 折线图 + 面积（Line + Area）
当需要强调"量"的概念时，在线下方添加半透明填充（opacity: 0.15）

### 多面板折线图（Small Multiples）
当系列 > 5 时，改用独立的面板图，每个面板一条折线，共用 X 轴

### 斜率图（Slope Chart）
当只有两个时间点、需要展示排名变化时使用
