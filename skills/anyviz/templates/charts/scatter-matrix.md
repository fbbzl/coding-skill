# 矩阵散点图（Scatter Plot Matrix / SPLOM）

## 适用场景
- 多个数值变量两两之间的关系探索
- 发现变量间相关性和聚类结构
- 对角线展示各变量的单变量分布
- 最佳变量数：3-8 个；数据点 50-5,000

## 数据格式
```json
{
  "variables": ["销售额", "市场占有率", "客户满意度", "成本"],
  "records": [
    { "销售额": 85000, "市场占有率": 45, "客户满意度": 8.5, "成本": 32000 },
    { "销售额": 62000, "市场占有率": 32, "客户满意度": 7.2, "成本": 28000 },
    { "销售额": 95000, "市场占有率": 52, "客户满意度": 9.1, "成本": 38000 }
  ],
  "categories": ["地区"],
  "values": ["华东", "华南", "华北"]
}
```

## 美学参数

```json
{
  "splom": {
    "point_radius": 3.5,
    "point_opacity": 0.65,
    "point_stroke_width": 1.0,
    "point_stroke_color": "#FFFFFF",
    "diagonal_opacity": 0.8,
    "diagonal_color": "#4269d0",
    "diagonal_bar_width_ratio": 0.6,
    "correlation_font_size": 9,
    "cell_gap": 8,
    "matrix_font_size": 10,
    "trend_line_width": 1.2,
    "trend_line_color": "#ff725c",
    "trend_line_dash": [6, 3],
    "trend_opacity": 0.5
  }
}
```

## 设计要点

1. **对角线设计**：对角线单元格（[i,i]）显示第 i 个变量的分布，可用直方图（条形宽度比 0.6）或密度曲线（#4269d0, 0.8 opacity），与散点图无缝对齐
2. **非对角线散点**：用 categorical 色板按分类变量着色；点半径 3.5px，opacity 0.65，白描边 1.0px，避免过度绘制
3. **对称填充与相关系数**：上三角放散点图+趋势线，下三角放相关系数热力图或放置同样的散点；相关系数文本 9px，位于单元格中心
4. **趋势线与关联性**：若两变量相关性明显（|r| > 0.3），自动绘制趋势线（#ff725c, 1.2px, dash [6,3], opacity 0.5）；高相关单元格用浅蓝背景突出
5. **坐标轴与刻度**：行列标签字号 10px，位置在矩阵边缘；每个小图的 X/Y 刻度标签 8px，仅在矩阵外侧显示，减少重复
6. **无障碍冗余编码**：颜色（按分类）+ 形状（可选小形状区分类别如三角/圆/方形）+ 交互提示（悬停显示具体坐标值和相关系数）

## 变体

### 相关系数矩阵（Correlation Heatmap SPLOM）
用热力图替代下三角，展示所有变量对的 Pearson/Spearman 相关系数，用色深表示强弱关系

### 混合 SPLOM（Mixed SPLOM）
上三角为散点+趋势线，下三角为相关系数数值或热力图，对角线为分布；提供完整的关联性和分布双重视图
