# 箱线图（Box Plot）

## 适用场景
- 展示数据分布的五数概括（最小值、Q1、中位数、Q3、最大值）
- 比较多个分组的分布差异和离散程度
- 快速识别异常值和分布的不对称性
- 最佳分组数：2-12 个；最佳数据点：100-50,000

## 数据格式
```json
{
  "groups": [
    { "name": "A部门", "values": [45, 52, 48, 55, 50, 62, 58, 60, 49, 95] },
    { "name": "B部门", "values": [68, 72, 70, 85, 75, 82, 78, 88, 71, 73] }
  ]
}
```

## 美学参数（从 default.json 继承）

```json
{
  "box": {
    "box_width_px": 30,
    "stroke_width_px": 1.5,
    "median_line_width_px": 2.5,
    "whisker_width_px": 1.0,
    "whisker_line_width_px": 1.0,
    "fill_opacity": 0.15,
    "outlier_radius_px": 4.5,
    "outlier_opacity": 0.6,
    "outlier_style": "x"
  }
}
```

## 设计要点

1. **箱体宽度与间距**：箱体宽度 30px，组间距 40-60px，与总组数成反比调整（12 个组时缩至 20px）
2. **颜色编码**：单组用 #4269d0，多组按 categorical 色板顺序分配，填充 opacity 0.15 避免过重
3. **须线标准**：默认 1.5 IQR，须线宽 1.0px，加号形端点（cap: round）
4. **中位数强调**：中位数线用 #333333，线宽 2.5px，视觉上突出数据集中趋势
5. **异常值标记**：× 符号标记（半径 4.5px，opacity 0.6），或用菱形 ◇，必须加标签避免色盲用户误读
6. **无障碍冗余编码**：分组用颜色+图例标签，异常值用形状（×）+数值标签，不仅靠颜色区分

## 变体

### 小提琴图（Violin Plot）
用密度曲线替代箱体，适合展示数据分布的完整形态，特别是多峰分布

### 蜂群图（Bee Swarm Plot）
在箱线图基础上叠加所有数据点的散点，点用 jitter 避免重叠，展示样本量和详细分布

### 带状图（Strip Plot）
去掉箱线，仅保留 jitter 散点，用于样本量较小（< 50）的轻量化分布展示

### 缺口箱线图（Notched Box Plot）
箱体两侧缩进形成"缺口"，缺口不重叠表示两组中位数有显著差异
