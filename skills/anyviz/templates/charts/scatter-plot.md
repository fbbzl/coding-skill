# 散点图（Scatter Plot）

## 适用场景
- 探索两个数值变量之间的相关性
- 识别数据聚类和异常值
- 最佳数据点：50-10,000

## 数据格式
```json
{
  "points": [
    { "x": 3.2, "y": 8.1, "category": "A" },
    { "x": 4.5, "y": 9.3, "category": "B" }
  ]
}
```

## 美学参数

```json
{
  "point": {
    "scatter": {
      "radius": 4.0,
      "opacity": 0.8,
      "stroke_width": 0.5,
      "stroke_color": "#FFFFFF"
    }
  }
}
```

## 设计要点

1. **透明度**：数据点较多时降低透明度（500 点 → 0.7, 2000 点 → 0.5, 10000 点 → 0.3）
2. **类别颜色**：按 categorical 色板分配分类颜色
3. **趋势线**：如果有明确的 x→y 关系，添加一条趋势线（#ff725c, dash [6,3]）
4. **异常值**：用特殊标记（如 "×"）标注统计异常值
5. **气泡大小**：如果引入第三个数值变量作为气泡大小，气泡面积应与值成比例（非直径）

## 变体

### 气泡图（Bubble Chart）
加入第三个变量映射到点的面积，第四变量可映射到颜色

### 六边形分箱图（Hexbin）
数据点 > 10,000 时使用，用六边形颜色深度表示点密度

### 边缘直方图（Marginal Histogram）
在散点图边缘附加每个变量的分布直方图，同时展示关系和分布
