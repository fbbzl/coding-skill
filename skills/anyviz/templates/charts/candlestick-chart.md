# 烛台图（Candlestick Chart）

## 适用场景
- 展示金融资产的 OHLC（开盘、最高、最低、收盘）价格数据
- 识别价格走势、波动率和涨跌模式
- 标准金融分析工具，用于股票、期货、加密货币
- 最佳数据点：20-500 个交易日/周期

## 数据格式
```json
{
  "dates": ["2024-01-01", "2024-01-02", "2024-01-03"],
  "ohlc": [
    { "open": 100, "high": 105, "low": 98, "close": 102 },
    { "open": 102, "high": 108, "low": 100, "close": 105 },
    { "open": 105, "high": 103, "low": 99, "close": 101 }
  ],
  "volume": [1200000, 1500000, 900000]
}
```

## 美学参数（从 default.json 继承）

```json
{
  "candlestick": {
    "candle_width_ratio": 0.7,
    "wick_width_px": 1.2,
    "candle_increasing_color": "#3ca951",
    "candle_decreasing_color": "#ff725c",
    "wick_color": "#333333",
    "border_width_px": 0.8,
    "volume_subplot_height_ratio": 0.25,
    "volume_opacity": 0.4
  },
  "axis": {
    "y_axis": { "label": "价格" }
  }
}
```

## 设计要点

1. **涨跌配色与地区差异**：中国金融市场习惯红涨绿跌，西方习惯绿涨红跌；建议默认采用西方标准（涨绿 #3ca951、跌红 #ff725c），并在文档中注明可配置
2. **烛体与影线**：烛体（矩形）宽度占 70% 的每日宽度，影线（竖线）宽 1.2px #333333；若收盘 > 开盘则烛体为绿实心，反之为红实心；无圆角
3. **成交量副图**：在主图下方显示成交量柱状图（高度约主图 25%），用同样的涨跌色 (#3ca951/#ff725c)，opacity 0.4，便于观察价格与成交量的关系
4. **Y 轴与参考线**：价格 Y 轴可不从零开始，但应标注关键价位（如开盘价、平均价）；添加均价参考线（#EE7B6F，虚线 [4,4]，opacity 0.7）
5. **X 轴标注**：显示日期或时间，重点标注月首和重要交易日；可配置显示频率
6. **无障碍设计**：若用户色盲，除颜色外用虚实线区分涨跌（涨实线、跌虚线），或在烛体内显示 ▲（涨）/ ▼（跌）符号

## 变体

### OHLC 条形图（OHLC Bar）
简化版，用单根竖线表示高-低范围，左侧小刻度表示开盘，右侧小刻度表示收盘，省空间

### 空心烛台（Hollow Candle）
烛体只绘制边框，不填充；涨时边框绿、内空白；跌时边框红、内填充，减少视觉重感

### 平均价与布林带（with Bollinger Bands）
在主图上叠加移动平均线（20/50/200 日）和布林带（标准差带），增强技术分析功能
