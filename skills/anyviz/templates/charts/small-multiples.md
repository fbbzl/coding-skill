# 小倍数图（Small Multiples）

## 适用场景
- 在统一坐标轴下比较多个类别、地区或时间段的相同指标
- 展示多个折线/柱状/散点图的平行对比，而非单一复杂图表
- 便于识别个体模式和群体趋势
- 最佳面板数：4-16 个

## 数据格式
```json
{
  "panels": [
    {
      "name": "区域 A",
      "x": ["2024-01", "2024-02", "2024-03"],
      "values": [120, 135, 148]
    },
    {
      "name": "区域 B",
      "x": ["2024-01", "2024-02", "2024-03"],
      "values": [80, 92, 105]
    }
  ],
  "chart_type": "line"
}
```

## 美学参数（从 default.json 继承）

```json
{
  "multiples": {
    "cols": 3,
    "gap_px": 28,
    "shared_x_axis": true,
    "shared_y_axis": true,
    "y_axis_label_frequency": "first_and_last"
  },
  "panel": {
    "title_font_size_px": 11,
    "title_weight": 400,
    "background_color": "#FFFFFF",
    "border_color": "#D0D0D0",
    "border_width_px": 0.5
  }
}
```

## 设计要点

1. **网格布局**：默认 3 列排列（4-9 个面板），10+ 个面板改为 4 列；面板间隙 28px；总体纵横比保持 1.6
2. **共享坐标轴**：所有面板共享同一 X、Y 轴范围与刻度，便于快速对比；Y 轴标签仅在左侧和右侧面板显示，中间面板隐藏
3. **面板标题**：标题字号 11px，400 字重，居中放在每个面板上方 10px；用类别名称或序号
4. **内容配色**：每个面板内容用 #4269d0；若面板代表不同分组（如部门）可用 categorical 色板给面板整体着色
5. **边框与分隔**：可选，若需要加边框用 #D0D0D0 0.5px；面板背景 #FFFFFF
6. **无障碍设计**：每个面板独立可读性强，避免色彩过度重复；若面板有层级关系，用不同饱和度 #4269d0 区分主次

## 变体

### 趋势面板（Trend Panels）
每个面板显示一条趋势线，同时显示原始数据点，专注于模式识别

### 地图小倍数（Map Multiples）
每个面板是一张地图，用热力颜色表示不同区域的指标值，同一背景地图叠加颜色

### 混合类型小倍数（Mixed Multiples）
同一布局中包含多种图表类型（柱状+折线+散点），展示多维度数据关系
