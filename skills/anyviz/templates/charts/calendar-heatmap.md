# 日历热力图（Calendar Heatmap）

## 适用场景
- 展示按日期的数值或活动频率，如 GitHub 贡献图、网站访问热度、每日销售
- 识别时间模式：周循环、季节性、节假日异常
- 适合展示一年或多年的时间序列数据
- 最佳数据跨度：1-3 年的每日数据

## 数据格式
```json
{
  "dates": ["2024-01-01", "2024-01-02", "2024-01-03", ...],
  "values": [5, 8, 3, 12, ...],
  "year": 2024
}
```

## 美学参数（从 default.json 继承）

```json
{
  "calendar": {
    "cell_size_px": 14,
    "cell_gap_px": 2,
    "cell_corner_radius_px": 2,
    "cell_stroke_width_px": 0.5,
    "cell_stroke_color": "#E8E8E8",
    "color_scale": "viridis",
    "no_data_color": "#F5F5F5",
    "layout": "week_rows"
  }
}
```

## 设计要点

1. **色板与映射**：用 Viridis 色板映射值的大小，从浅 (#F7F7F7) 到深 (#440154)；无数据格用 #F5F5F5；支持对数色标处理跨度大的数据
2. **单元格设计**：每格 14px × 14px，圆角 2px，格间距 2px，细边框 0.5px #E8E8E8 分隔；鼠标悬停时显示日期和具体数值（tooltip）
3. **周与月的布局**：从左到右按周排列（纵向为周一到周日 7 行），月份标题在每月首行左侧（11px 字号）；年份标题在顶部
4. **年份展示**：单年完整日历为主；多年对比时，叠加展示 2-3 年，每年占一行或独立面板
5. **参考线与注解**：可添加不同月份间的细竖线（0.5px #D0D0D0）分隔；关键日期或节假日可加标注
6. **无障碍设计**：颜色外加文本标签显示活动频率等级（Low/Medium/High），或在单元格内显示简单数字

## 变体

### 年历视图（Year Overview）
多年堆叠展示，每行一年，纵向比较年度变化

### 星期×小时热力图（Day-Hour Heatmap）
改为星期（横轴）×小时（纵轴），展示一周内每小时的活动热度，适合分析工作日模式

### 月日热力图（Month Detail）
细化到单个月份，每格代表一天，加入周标注和周末高亮
