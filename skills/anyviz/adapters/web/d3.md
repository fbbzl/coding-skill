# D3.js 适配器

## 适用场景
- 需要像素级精确控制的定制化可视化
- ECharts 等库不支持的图表类型（如自定义网络图、特殊投影地图）
- 动态数据更新和过渡动画
- 数据艺术/创意可视化

## 核心映射：美学 → D3 代码

### 颜色
```javascript
const colorPalette = ['#4269d0', '#3ca951', '#ff725c', '#a463f2', '#efb118',
                       '#6cc5b0', '#9696a0', '#f5a623', '#ca5bb8', '#ff8ab7'];

// 分类色板比例尺
const colorScale = d3.scaleOrdinal(colorPalette);

// 顺序色板
const sequentialScale = d3.scaleSequential(d3.interpolateBlues)
  .domain([minValue, maxValue]);
```

### SVG 画布设置
```javascript
const margin = { top: 40, right: 30, bottom: 50, left: 60 };
const width = 800 - margin.left - margin.right;
const height = 500 - margin.top - margin.bottom;

const svg = d3.select('#chart')
  .append('svg')
  .attr('width', width + margin.left + margin.right)
  .attr('height', height + margin.top + margin.bottom)
  .style('background-color', '#FFFFFF')
  .style('font-family', "'Helvetica Neue', Helvetica, Arial, 'PingFang SC', 'Microsoft YaHei', sans-serif")
  .append('g')
  .attr('transform', `translate(${margin.left},${margin.top})`);
```

### 排版
```javascript
// 标题
svg.append('text')
  .attr('class', 'chart-title')
  .attr('x', 0)
  .attr('y', -margin.top + 24)
  .attr('font-size', 16)     // h1
  .attr('font-weight', 600)
  .attr('fill', '#1A1A1A')
  .text('Chart Title');

// 轴标题
svg.append('text')
  .attr('class', 'axis-label')
  .attr('font-size', 11)     // h3
  .attr('fill', '#333333')
  .attr('text-anchor', 'middle');

// 刻度标签
g.append('g')
  .attr('font-size', 10)     // h4
  .attr('fill', '#666666')
  .call(d3.axisBottom(xScale));
```

### 线条
```javascript
// 折线
const line = d3.line()
  .x(d => xScale(d.x))
  .y(d => yScale(d.y))
  .curve(d3.curveMonotoneX);   // 平滑曲线

svg.append('path')
  .datum(data)
  .attr('fill', 'none')
  .attr('stroke', colorScale(0))
  .attr('stroke-width', 2)     // line width
  .attr('stroke-linecap', 'round')
  .attr('stroke-linejoin', 'round')
  .attr('d', line);

// 网格线
d3.axisBottom(xScale)
  .tickSize(-height)
  .tickFormat('');

svg.append('g')
  .call(/* ... */)
  .attr('stroke', '#E0E0E0')
  .attr('stroke-width', 0.5)
  .attr('stroke-dasharray', '2,2');
```

### 散点
```javascript
svg.selectAll('.dot')
  .data(points)
  .enter()
  .append('circle')
  .attr('r', 4)               // scatter radius
  .attr('fill', d => colorScale(d.category))
  .attr('fill-opacity', 0.8)   // scatter opacity
  .attr('stroke', '#FFFFFF')
  .attr('stroke-width', 0.5);
```

### 柱状图
```javascript
const barWidth = xScale.bandwidth() * 0.7;  // width_ratio

svg.selectAll('.bar')
  .data(data)
  .enter()
  .append('rect')
  .attr('x', d => xScale(d.category) + xScale.bandwidth() * 0.15)
  .attr('width', barWidth)
  .attr('fill', colorScale(0));
```

## D3.js 注意事项

1. **数据绑定**：使用 `.join()` 替代 `.enter().merge()` 以正确处理 enter/update/exit
2. **过渡动画**：默认使用 `transition().duration(800).ease(d3.easeCubicOut)`
3. **响应式**：使用 `viewBox` 而非固定宽高，或监听 resize 事件重绘
4. **无障碍**：为 SVG 元素添加 `<title>` 和 `aria-label`
5. **性能**：数据点 > 5000 时考虑 Canvas 渲染替代 SVG
