## 3. Execution — Methods & Tools

### Task 1: Fix chart on mobile

**Files modified:** `src/components/charts/RevenueChart.tsx`

**Root cause:** `ResponsiveContainer` had a fixed `height={400}` that overflowed on small screens, causing the SVG to render outside the visible area.

```tsx
// Before
<ResponsiveContainer width="100%" height={400}>

// After
<ResponsiveContainer width="100%" height="100%" minHeight={200}>
```

**Decision log:** Used `minHeight` instead of a breakpoint-conditional height — keeps the component self-contained without needing to pass props from the parent.

---

### Task 2: Fix CSV export 500 error

**Files modified:** `src/app/api/export/route.ts`

**Root cause:** `date.toLocaleDateString()` throws when the database returns `null` for orders with no completion date. The API route had no null guard.

```ts
// Before
const row = [order.id, order.date.toLocaleDateString(), order.total]

// After
const row = [order.id, order.date ? order.date.toLocaleDateString('en-US') : 'N/A', order.total]
```

---

### Task 3: Sidebar active link

**Files modified:** `src/components/layout/Sidebar.tsx`

**Approach:** Added `usePathname()` from `next/navigation` to check current route against each nav item's `href`.

```tsx
const pathname = usePathname()
<Link className={pathname === item.href ? 'bg-blue-600 text-white' : 'text-gray-400 hover:text-white'}>
```
