# HUMAN-AI - Движок очеловечивания текста

\[ [English](README.md) | **Русский** \]

> **Если читатель забыл, что к тексту прикасалась машина - вы победили.**

AI skill-система, которая переписывает машинно-сгенерированный текст так, чтобы он звучал как человеческий - на 9 языках. Обнаруживает и удаляет AI-отпечатки через 5-стадийный пайплайн.

---

## Что это?

**HUMAN-AI** - это системный промпт (skill) для любой LLM: GPT, Claude, Gemini, DeepSeek, open-source моделей. Он превращает AI-текст, который «пахнет машиной», в текст, который читается так, будто его написал компетентный человек.

AI пишет с отпечатком: одинаковый ритм, одинаковая структура, одни и те же сожжённые слова, одна и та же мёртвая идеальность. HUMAN-AI стирает этот отпечаток - не делая текст «менее AI-шным», а делая его *более человеческим*.

---

## Как работает

1. **Загрузите `SKILL.md`** как системный промпт в любую LLM.
2. **Дайте задачу:** «Перепиши чтобы звучало по-человечески. Язык: ru.» / «Сделай из этого кейс. EN.» / «Просто вычисти AI-паттерны.»
3. **Получите результат** с определённым языком, установленным тоном, пройденными стадиями и чейнджлогом.

**Три режима:**

- **Полный пайплайн** - 5 стадий: зачистка → конкретика → тон → ритм → вычитка. Гибкий: стадии можно пропускать с обоснованием.
- **Одна стадия** - Только зачистка, только конкретика, только ритм.
- **Режим аудита** - Только диагностика. Без правок. Показать все найденные AI-паттерны.

---

## 5-стадийный пайплайн

```
зачистка → конкретика → тон → ритм → вычитка
```

| Стадия | Что делает | Пропустить если |
|--------|-----------|-----------------|
| 1. Зачистка | Удаление AI-паттернов: зачины, сожжённые слова, фейковые переходы, хеджирование, концовки | Нет AI-паттернов |
| 2. Конкретика | Замена абстракций на цифры, примеры, детали | Все утверждения уже rung 2+ |
| 3. Тон | Настройка голоса (7 профилей) | Тон уже верный |
| 4. Ритм | Ломаем метроном: длина предложений, зачины, фрагменты | Ритм уже разнообразный |
| 5. Вычитка | Финальное сканирование на остатки AI | Всегда (минимальное) |

## 7 тоновых профилей

| Тон | Голос | Для чего |
|-----|-------|----------|
| `expert` | Практик | Тех. документация, глубокий анализ |
| `biz` | Консультант | B2B-предложения, страницы услуг |
| `human` | Умный друг | Блог, «о нас», письма |
| `social` | Скроллер | LinkedIn, Telegram, Twitter |
| `landing` | Продавец | Страницы продуктов, лендинги |
| `article` | Объяснитель | Лонгриды, гайды, туториалы |
| `case` | Кейс-стади | Портфолио, истории успеха |

Каждый тон имеет языковые маркеры: частоту фрагментов, правила союзов, уровень формальности, культурные ноты.


### Конкретная задача (сценарий)
```

```

### Одна стадия
```
Только вычисти AI-паттерны.
Добавь конкретики. Тон не меняй.

```

### Только аудит
```
Скажи что не так с текстом. Не переписывай.
```

### Исправление перевода
```
Это переведено с русского на английский. Сделай чтобы звучало как родное.
```

## Пример

**Запрос:** «Перепиши это вступление AI-статьи чтобы звучало по-человечески. RU.»

**Вход:**
> «В современном быстро меняющемся цифровом ландшафте компании всё чаще используют передовые AI-решения для оптимизации рабочих процессов и повышения эффективности. Данное комплексное руководство рассматривает трансформационную силу искусственного интеллекта...»

**Выход:**
```
[LANG: ru]
[TONE: human]
[PIPELINE: cleanup → specificity → tone → rhythm → proofread]

Большинство AI-инструментов обещают экономию времени. Не экономят. Просто перекладывают работу из одного места в другое.

Мы протестировали двенадцать штук за три месяца. Четыре реально помогли. Остальные добавили совещаний, дашбордов и новый канал в Slack, который вы не просили.

Вот что общего у тех четырёх. И почему остальные восемь не взлетели.

---
[CHANGELOG]
- Удалён зачин «В современном быстро меняющемся...»
- Сожжённые слова: передовые, оптимизация, эффективность, данное, комплексное, трансформационный
- Конкретика: «двенадцать штук / три месяца / четыре помогли»
- Тон: human - прямой, с мнением, фрагмент
- Ритм: 9сл → 9сл → 4сл → 20сл → 10сл → 10сл → 13сл
```

*→ Больше примеров в `examples/`*

## Требования

- Любая LLM с полем системного промпта
- Для полной функциональности: skill-система с загрузкой файлов из папки (OpenCode, Claude Code)
- Автономно: скопируйте `SKILL.md` как системный промпт - пайплайн полный
- Без API-ключей, без инструментов, без зависимостей - чистый промпт-инжиниринг

## Этика

**Всегда:**
- Помечайте придуманные цифры флагом `[ПРОВЕРИТЬ]`
- Сохраняйте факты - меняйте только подачу
- Декларируйте все изменения в чейнджлоге

**Никогда:**
- Не придумывайте факты, статистику, отзывы, имена клиентов
- Не исправляйте фактические ошибки молча (выносите в [FACTUAL NOTES])
- Не переписывайте юридические, медицинские и safety-critical тексты

## Интеграция с MindFluence

HUMAN-AI работает в паре с [MindFluence](https://github.com/MADEVAL/MindFluence) (когнитивные искажения в маркетинге):

- MindFluence генерирует убедительный текст с психологическими триггерами
- HUMAN-AI очеловечивает его, не ломая структуру искажений
- Маппинг тонов: `warm-human` → `human`, `expert-calm` → `expert`, `bold-sell` → `landing`
- Используйте `PIPELINE: cleanup → specificity → tone(skipped) → rhythm → proofread` чтобы сохранить тон MindFluence
