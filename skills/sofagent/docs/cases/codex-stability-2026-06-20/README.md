## Codex 安装踩坑（2 个 bug）

### Bug 1 · install.sh Codex 分支变量未初始化（P0）

- `install.sh` 只在 WorkBuddy 和 OpenClaw 路径初始化 `SOFAGENT_DATA`
- Codex 汇总阶段仍执行 `echo "$SOFAGENT_DATA"`，在 `set -u` 下导致退出
- 临时处理：显式传入 `SOFAGENT_DATA=<项目路径>/.sofagent`，并指定 `--project-dir`

### Bug 2 · verify.sh 误查 OpenClaw Hook（P0）

- `verify.sh --platform=codex` 正确识别 `~/.codex`，但后续仍检查 `hooks/sofagent-load-chain/` 和 `openclaw.json`
- Codex 手动平台本来不部署该 Hook，因此会产生稳定误报并返回失败
- 复测结果：14 PASS / 7 WARN / 1 FAIL，退出码 1
