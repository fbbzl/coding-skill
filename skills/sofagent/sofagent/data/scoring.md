# scoring.md · Skills 评分记录

> 运行时由 skill-iterate 子 Skill 自动维护。树形加载，按任务类型分分支。
> 每个叶子文件存社区评分、使用次数、个人备注。Loop Agent（closure 模式）按 §十 复盘，结果汇入 orchestrator/。
>
> 本文件是格式参考——实际目录在 Skill 首次加载时自动初始化。
>
> 数据来源：task/logs/。读取方：主 Agent 选 Skill 时按需查询；反思时读取，汇入 think.md。
>
> ⚠️ 以下路径为相对于 `{SOFAGENT_DATA}/` 的简写。实际读写时请加 `{SOFAGENT_DATA}/` 前缀。
>
> ⚠️ Skill 的 `description` 字段不是给人看的摘要——是给模型判断是否调用该 Skill 的**触发条件**。写错 description = Skill 永远不会被触发。写「什么时候用」，不要写「它是干什么的」。
